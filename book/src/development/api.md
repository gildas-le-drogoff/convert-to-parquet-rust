```bash

```

## Main Modules

| Module       | Description                                          |
| ------------ | ---------------------------------------------------- |
| `conversion` | Conversion pipeline: orchestration, reading, writing |
| `analysis`   | Block analysis, `RecordBatch` construction, metrics  |
| `schema`     | Type inference via sampling                          |
| `utils`      | Delimiter/header detection, date parsing             |
| `inspect`    | Parquet metadata inspection                          |
| `json`       | JSON → intermediate CSV conversion                   |
| `xlsx`       | Spreadsheet → intermediate CSV conversion            |
| `to_csv`     | Reverse Parquet → CSV conversion                     |
| `to_jsonl`   | Reverse Parquet → JSONL conversion                   |
